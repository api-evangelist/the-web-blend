# The Web BlendThis is a repo for managing machine readable artifacts for the [The Web Blend APIs](http://www.thewebblend.com/api). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [The Web Blend APIs](http://www.thewebblend.com/api).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.thewebblend.com/api)- [Developer](http://www.thewebblend.com/api)- [Twitter](https://twitter.com/#!/TheWebBlend)The [APIs.json](https://github.com/api-evangelist/the-web-blend/blob/master/apis.json) index provides links to all the relevant links for the The Web Blend APIs, including the OpenAPI and Postman if they are present.
